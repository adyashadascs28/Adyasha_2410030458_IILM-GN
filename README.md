## 📌 Description
This project predicts whether a person has diabetes using Machine Learning. The model is built using the Support Vector Machine (SVM) algorithm and trained on the Pima Indians Diabetes Dataset. It predicts the likelihood of diabetes based on medical attributes such as glucose level, BMI, age, etc.

---

## 🤖 Algorithm Used
- Support Vector Machine (SVM)

---

## 📊 Dataset
Pima Indians Diabetes Dataset

## ▶️ How to Run the Project

### Step 1: Install Required Libraries
pip install pandas numpy scikit-learn matplotlib streamlit

### Step 2: Run model file
py diabetes_model.py

### Step 3: Run web app
py -m streamlit run app.py

After running, open the link shown in terminal (usually http://localhost:8501)
